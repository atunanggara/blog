# Learning Data Science

A data science blog to showcase my work, built with [Quarto](https://quarto.org/).

Migrated from fastpages following the [nbdev migration guide](https://nbdev.fast.ai/tutorials/blogging.html#migrating-from-fastpages).

## Local development

```
quarto preview
```

## Publishing

```
quarto publish gh-pages
```
